Checks if a character is printable.

  • parameter c

    The character to check.

  • returns

    true if c is printable, false otherwise.

Checks if all characters in a string are printable.

  • parameter s

    The string to check.

  • returns

    true if all characters in s are printable, false otherwise.

val template : string -> (string -> string) -> string

Fills a template string using values from a provided map. Template variables are denoted by $identifier. Dollar signs can be escaped with $$.

  • parameter template

    The template string containing placeholders.

  • parameter map

    A function mapping identifiers to replacement strings.

  • returns

    The template string with placeholders replaced by mapped values.

val capitalize : string -> string

Capitalizes the first letter of a string and lowercases the rest.

  • parameter s

    The input string.

  • returns

    A string with the first letter capitalized and the rest in lowercase.

val center : ?fill_char:char -> string -> int -> string

Centers a string in a field of a given width using a specified fill character.

  • parameter s

    The string to center.

  • parameter w

    The target width.

  • parameter fill_char

    The character used for padding (default is space).

  • returns

    The centered string.

val ljust : ?fill_char:char -> string -> int -> string

Left-justifies a string by padding it on the right with a specified character.

  • parameter s

    The string to justify.

  • parameter w

    The total width of the resulting string.

  • parameter fill_char

    (optional) The character used for padding. Defaults to a space.

  • returns

    The left-justified string. If s is longer than w, s is returned unchanged.

val rjust : ?fill_char:char -> string -> int -> string

Right-justifies a string by padding it on the left with a specified character.

  • parameter s

    The string to justify.

  • parameter w

    The total width of the resulting string.

  • parameter fill_char

    (optional) The character used for padding. Defaults to a space.

  • returns

    The right-justified string. If s is longer than w, s is returned unchanged.

val partition : string -> string -> string * string * string

Splits a string s into a 3-tuple based on the first occurrence of the separator sep.

The result is a tuple (before, sep, after), where:

  • before is the substring before the first occurrence of sep.
  • sep is the separator itself (if found).
  • after is the substring after the first occurrence of sep.

If the separator sep is not found, returns (s, "", "").

  • parameter s

    The input string to partition.

  • parameter sep

    The separator string to search for.

  • returns

    A tuple (before, sep, after).

val remove_prefix : string -> string -> string

Removes the specified prefix from the string s if it starts with prefix.

If s does not start with prefix, returns s unchanged.

  • parameter s

    The input string.

  • parameter prefix

    The prefix to remove.

  • returns

    The string without the prefix if present, otherwise the original string.

val remove_suffix : string -> string -> string

Removes the specified suffix from the string s if it ends with suffix.

If s does not end with suffix, returns s unchanged.

  • parameter s

    The input string.

  • parameter suffix

    The suffix to remove.

  • returns

    The string without the suffix if present, otherwise the original string.
